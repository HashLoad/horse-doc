Instalação

Boss

A forma mais fácil e recomendada de instalar o Horse é com Boss, assim é possível gerenciar a versão e também adicionar automaticamente à Library Path do Delphi.

$ boss install github.com/HashLoad/horse

Manual

Faça o Download e inclua o caminho correto da pasta src/ na Library Path do Delphi. Abra o pacote horse.dpk e compile-o.
