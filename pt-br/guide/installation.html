Instalação

Boss

A forma mais fácil e recomendada de instalar o Horse é com Boss. Com ele é possível gerenciar a versão das dependências utilizadas em sua API, além de configurar automaticamente o Library Path do seu projeto.

Para realizar a instalação, basta abrir o terminal de sua preferência e digitar os seguintes comandos:

OBS: vale ressaltar que o comando boss init é para inicializar o gerenciador de dependências e deve ser executado apenas uma vez. Caso já tenha feito, não é necessário executar o mesmo novamente.

Manual

Faça o download da última versão do Horse e inclua a pasta src/ no Library Path do seu projeto.
